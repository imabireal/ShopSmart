# ShopSmart - E-commerce Web App

A Flask-based e-commerce web application with product Recommendation using apriori algorithm, user authentication, product catalog, shopping cart, and checkout functionality.

## Features
- User registration and login
- Product browsing and catalog
- Shopping cart management (add, update, remove items)
- Buy now functionality for immediate purchase
- Secure checkout process
- Session-based cart persistence
